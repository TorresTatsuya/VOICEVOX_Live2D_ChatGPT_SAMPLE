# VOICEVOX_Live2D_ChatGPT_SAMPLE

# 概要
VOICEVOX + Live2D + ChatGPTを使った対話のWEBアプリです。
ChatGPTをしゃべらせてみたいという理由で作ったものです。
キャラに関するプログラムはLive2D WEB SDKを使って事前にコンパイルしています。
